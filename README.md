# Flask Basics Learning Project

This repo contains my practice project while learning **Flask**, following tutorials and coding along to understand web development with Python.
Tutorial can be found here [[freeCodeCamp](https://youtu.be/Z1RJmh_OqeA?si=Q4045en1fZQF5fBX)]

## Project Overview 
  - Learning the fundamentals of Flask:
  - Routes and views
  - Templates using Jinja2
  - Forms with Flask-WTF
  - Database integration with SQLAlchemy
  - Experimenting with concepts from coding tutorials

## How to Run
1. Activate your virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `flask run`
4. Open `http://127.0.0.1:5000` in your browser

Notes:
- `venv/` is ignored
- Database is local
- This was for learning purposes only
